# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-07-10

- Unified the shared execution pipeline with normalized output/error handling and built-in tool history capture.
- Deepened structured data workflows with JSON -> YAML conversion, follow-up actions, and clearer XML diagnostics.
- Added a new text diff tool for text and file comparison, including git-style and side-by-side diff views.

## [1.0.0] - 2026-07-03

- Initial public release preparation.
- Includes JSON, Base64, timestamp, YAML, XML, random string, Cron, URL, Hash, JWT, UUID, and regex tools.
